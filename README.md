# Deep Learning Fundamentals

### ANN · CNN · LSTM · Keras · TensorFlow · Google Colab

Early deep learning experiments covering the core building blocks of modern neural networks — from activation functions and fully connected layers to convolutional image classifiers and recurrent sequence models. All notebooks were built and run on Google Colab.

---

## Notebooks

| Notebook | Topic |
|---|---|
| `Activation_Functions_and_Building_ANN` | Activation functions (ReLU, sigmoid, softmax), building and training an ANN with Keras |
| `CNN_CIFAR10_Image_Classifier` | CNN trained on CIFAR-10 for 10-class image classification |
| `CNN_Deep_Dive` | Deeper exploration of CNN components — filters, pooling, receptive fields |
| `Convolutional_Neural_Network_(Using_TensorFlow)` | CNN implementation from scratch using TensorFlow |
| `LSTM_airline_passengers` | LSTM for time series forecasting on the classic airline passengers dataset |
| `sine_wave_LSTM` | LSTM trained to learn and predict a sine wave sequence |

---

## Topics Covered

- **ANN** — Dense layers, activation functions, loss functions, optimizers
- **CNN** — Convolution, pooling, feature maps, image classification
- **LSTM** — Sequence modeling, time series forecasting, vanishing gradient problem
- **Keras / TensorFlow** — Model building, training loops, evaluation
